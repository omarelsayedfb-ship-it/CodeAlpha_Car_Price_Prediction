# CodeAlpha_Car_Price_Prediction

## 📌 Project Overview
This repository contains my submission for **Task 3: Car Price Prediction with Machine Learning** as part of my **Data Science Internship at CodeAlpha**.

The goal of this project is to train a machine learning regression model to predict car prices based on various car-related features such as brand goodwill, horsepower, mileage, engine size, and other technical specifications.

## 🚀 Features & Workflow
- **Data Preprocessing:** Handling missing values, encoding categorical features, and scaling numerical variables.
- **Feature Engineering:** Extracting relevant insights to improve model performance.
- **Exploratory Data Analysis (EDA):** Visualizing data distributions, correlations, and price trends using Matplotlib and Seaborn.
- **Model Training & Evaluation:** Training regression models (e.g., Linear Regression, Decision Trees, or Random Forest) using Scikit-learn and evaluating them with metrics like $R^2$ Score and Mean Absolute Error (MAE).

## 🛠️ Technologies Used
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## 📂 Project Structure
- `car_price_prediction.ipynb` — The Jupyter Notebook containing the complete end-to-end workflow (Data Cleaning, EDA, Model Training, Evaluation).
- `car_data.csv` — The dataset used for training and testing the model.
- `README.md` — Project documentation.
